# NewHack 🚀

Website institucional da **NewHack** — ecossistema de apoio ao empreendedor, focado em acelerar o crescimento de startups em estágio inicial.

## 🌐 Sobre

A NewHack combina educação empreendedora, capital de risco e inovação aberta, criando oportunidades para empreendedores se conectarem com líderes, investidores e empresas de ponta.

## 📁 Estrutura do Projeto

```
newhack/
```

## 🛠️ Tecnologias

- **HTML5** — Estrutura semântica
- **CSS3** — Estilização com variáveis CSS, Flexbox e Grid
- **Google Fonts** — Tipografia (Questrial)

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/newhack.git
   ```
2. Abra o `index.html` no navegador ou utilize um servidor local (ex: Live Server no VS Code).
